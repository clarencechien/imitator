1. 宏大願景：AI 作為統一服務

本節探討 Sam Altman 的核心目標：將 AI 打造成一個無所不在的個人化助手，類似於電晶體或作業系統，滲透到生活的各個層面。這個統一願景是驅動 OpenAI 一切行動的根本動力。

本節重點

  • OpenAI 的終極目標是打造一個單一、連貫的 AI 服務，陪伴使用者一生。
  • Altman 將 AI 比作「電晶體」，強調其無所不在的滲透力。
  • 此願景涵蓋產品 (ChatGPT)、平台 (API) 和底層研究 (AGI)。

核心理念

Altman 構想的 AI 是一個統一的助手，它不僅存在於 ChatGPT 中，更會透過 API 延伸至所有第三方應用，提供無縫的個人化體驗，真正了解並協助使用者。

Altman 的類比：電晶體

他最喜歡的歷史類比是「電晶體」。AI 不會是一個單一產品，而是會像電晶體一樣，成為所有消費性和企業產品中不可或缺的一部分，無形地提升一切。

關鍵引言

"我認為它會像水一樣滲透到每一個消費性產品和企業產品中。"

2. 基礎設施：一場史無前例的豪賭

為了實現宏願，OpenAI 正在進行一場價值上兆美元的基礎設施建設。本節解析其與供應鏈巨頭的合作、獨特的融資策略，以及為何這場建設必須是「同時並行」而非「依序進行」。

本節重點

  • AI 建設是多層次的，從電力、晶圓廠到晶片和記憶體，環環相扣。
  • Altman 的投資者背景讓他擅長進行大規模的資本配置與風險評估。
  • OpenAI 計劃協助合作夥伴融資，以加速整個產業鏈的建設。

互動式問答：建設藍圖

最大的挑戰在於「同時性」。Altman 強調，你不能先建好發電廠，再等晶圓廠，再等晶片。電力、設施、晶圓產能、機架、消費者需求、商業模式等所有環節都必須同時並行發展，這需要巨大的協調和資本投入。

最終將由 OpenAI 的營收支付。但在營收實現之前，OpenAI 計劃扮演「財務擔保人」的角色，利用其市場地位和保證採購協議，協助供應鏈上的合作夥伴（如晶片、記憶體製造商）以更優惠的條件獲得融資。

3. 產品策略：以信任為核心的平台

ChatGPT 的爆紅為 OpenAI 提供了建立平台的絕佳機會。本節聚焦其產品策略的演進，從失敗的 Plugins 到成功的 Apps，以及為何「信任」是其商業模式中不可動搖的基石。

本節重點

  • 產品策略的核心是打造一個開放、共榮的生態系，而非封閉花園。
  • OpenAI 堅持使用者與第三方應用的直接關係，以扶植新創。
  • 「信任」高於短期商業利益，因此不會在核心問答中進行競價排名。

從 Plugins 到 Apps

Altman 坦承早期的 Plugins 並不成功，但 GPTs 在特定工作流程中找到了用戶。最新的 Apps 策略，是基於過去的經驗，更強調讓合作夥伴控制 UI 和使用者關係，打造一個真正共榮的平台。

信任的價值

Altman 認為，使用者之所以喜愛 ChatGPT，是因為相信它真心在幫忙。一旦為了廣告或商業利益而犧牲答案的公正性，這種信任就會崩潰。因此，任何商業模式都必須以不損害使用者信任為前提。

4. Sora & 創意革命

Sora 大幅降低了影片創作的門檻，挑戰了網路內容創作的傳統法則。本節深入探討 Sora 如何釋放人類潛在的創作需求，並分析其背後的產品定位與獨特的商業模式考量。

本節重點

  • Sora 的成功在於降低「創作活化能」，讓更多人能將想法變為現實。
  • Sora 的高算力成本意味著傳統廣告模式可能不適用，付費生成將是選項。
  • Sora 作為獨立 App，是為了區隔其「娛樂/社交」屬性與 ChatGPT 的「個人/私密」屬性。

背景知識：90/9/1 法則

這是描述網路社群參與度的經驗法則：

  • 90% 的使用者是「潛水者」，他們只瀏覽內容而不參與。
  • 9% 的使用者是「貢獻者」，他們會進行少量互動，如評論或按讚。
  • 1% 的使用者是「創作者」，他們產出絕大部分的原創內容。

Sora 的出現，可能透過降低創作門檻，從根本上改變這個比例分佈。

5. 回饋與未來展望

在高速發展的同時，OpenAI 也面臨版權、使用者回饋和資源限制等挑戰。本節探討 OpenAI 如何在創新與責任間尋求平衡，並根據複雜的市場信號快速調整策略。

本節重點

  • OpenAI 預測，版權方將從抵制 AI 轉向擁抱 AI，尋求更多曝光。
  • 公司需平衡「專業使用者」和「普通大眾」兩種截然不同的需求。
  • 快速聆聽數據和質化回饋，並迅速迭代，是 OpenAI 的核心策略。

版權的共同演進

Altman 認為，權利方對影片的反應比圖像更強烈。但他預測，隨著時間推移，權利方會意識到 AI 能加深與粉絲的連結，最終的訴求將從「不准用我的內容」變為「請公平地多用我的內容」。

平衡兩類使用者

從 GPT-5 的回饋中，Altman 學到最大的教訓是：Twitter 上的 AI 重度使用者（Power User）和廣大普通使用者（Normies）的需求截然不同。打造能同時滿足兩者的產品是巨大挑戰，需要認真對待。
